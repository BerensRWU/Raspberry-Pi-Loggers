# Sense-Hat-Camera-Logger

## Requirements
The repo was tested on a Raspberry Pi Version 4.
The following packages are required:
 * SenseHat
 * PiRGBArray
 * PiCamera
 * pyaudio
 * wave
 * gps
 * OpenCV
